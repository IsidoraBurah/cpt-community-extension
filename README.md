### **CPT Community Extension** — это open-source расширение для сайта ЧПТ

---
## Как локально запустить расширение
1) Откройте браузер Google Chrome 
2) В адресной строке введите: `chrome://extensions`
3) В правом верхнем углу включите переключатель **«Режим разработчика»** (Developer mode)
4) Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked)
5) Выберите папку cpt-community-extension

После загрузки расширение появится в списке.
